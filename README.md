# OndeComprar

Introdução
Este programa tem como objetivo manter um catálogo atualizado dos produtos de informática disponíveis em estoque nas lojas físicas da cidade;
O $nome_do_programa surge da necessidade dos consumidores terem a informação de onde existe determinado produto e qual o seu preço, para com isto não perder tempo procurando e muitas vezes voltando de mãos vazias para a sua casa.

Objetivos do Sistema
Geral
Oferecer um serviço inovador para pernambuco, evitando a situação frequente de ter que procurar de loja em loja por algum produto que muitas vezes poderia ser encontrado no bairro vizinho.
Específicos
Com a implementação do $nome_do_programa pretende-se alcançar os seguintes objetivos em relação aos:
 Empresa
	• Manter cadastro de produtos;
	• Atualizar dados do Produto;
 Login Senha
	• Manter cadastro de login e Senha;
	• Consultar e alterar e Excluir login e Senha

Interface Gráfica (usuário)
	• Consultar Produto;
	• Consultar Loja;


Classes e Atributos
Loja:
	• new Endereço;
	• new Produto;
Produto:
	• codigo;
	• nome;
	• descricao;
	• preco;
Métodos
adicionarProduto;
removerProduto;
buscarProduto;
